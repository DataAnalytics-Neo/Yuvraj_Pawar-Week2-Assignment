# Excel Assignments – Week 2

## Overview
This repository contains completed Excel assignments for **Week 2**. The tasks were originally provided in `spreadsheet.xlsx`, and all solutions have been compiled in `Week2-Task.xlsx`.  

The assignments cover various Excel functionalities including formulas, functions, conditional statements, data analysis, and chart visualizations. This repository serves as a reference for practicing Excel tasks efficiently.

## Files in the Repository

1. **spreadsheet.xlsx** – Contains the original tasks and exercises assigned.  
2. **Week2-Task.xlsx** – Contains completed solutions for all tasks from `spreadsheet.xlsx`.  
3. **README.md** – Provides an overview and explanation of the tasks and files.

## Tasks Overview

The Week 2 Excel assignments cover a variety of common Excel tasks and formulas. 
**Exercise 1** involves calculating the total and average of numeric data using the SUM and AVERAGE functions. 
**Exercise 2** focuses on computing average scores, possibly per student, to assess performance. 
**Exercise 3** uses an IF condition to determine bonus eligibility based on specific criteria. 
**Exercise 4** employs COUNT functions to count the number of entries or occurrences meeting certain conditions. 
**Exercise 5** applies SUMIF to sum values conditionally based on given criteria. 
**Exercise 6** demonstrates VLOOKUP or HLOOKUP to look up and retrieve information from a table using a key value. 
**Exercise 7** combines nested formulas, using IF statements along with arithmetic calculations to handle multiple conditions. Finally, 
**Exercise 8** calculates commission as a percentage of sales, optionally using conditional logic for different scenarios.  

> **Note:** All tasks have been solved in `Week2-Task.xlsx`, with formulas applied wherever required.
